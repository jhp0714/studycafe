# Pass

| 필드 | 타입 | 제약/설명 |
| --- | --- | --- |
| id | pk |  |
| user_id | fk(User) |  |
| product_id | fk(Product) |  |
| order_id | fk(Order) | UNIQUE |
| pass_kind | enum | time, flat, fixed, locker |
| status | enum | active, expired, canceled |
| start_at | datetime | 결제 완료 후 |
| end_at | datetime | 기간제 상품 끝나는 날짜 |
| remaining_minutes | int | 시간제 상품 남은 시간 분으로 관리 |
| fixed_seat_id | FK(Seat) | fixed 전용, 그 외 null |
| locker_id | fk(locker) | locker 전용, 그 외 null |
| created_at | datetime |  |

**fixed/locker 이동 시**: `Pass.fixed_seat_id(or locker_id)`와 `SeatUsage.seat_id(or LockerUsage.locker_id)`를 **같은 트랜잭션에서 같이 업데이트**하고 Logs 남긴다.

- CHECK(pass_kind='fixed' ⇒ fixed_seat_id NOT NULL AND locker_id IS NULL)
- CHECK(pass_kind='locker' ⇒ locker_id NOT NULL AND fixed_seat_id IS NULL)
- CHECK(pass_kind IN ('time','flat') ⇒ fixed_seat_id IS NULL AND locker_id IS NULL)

## 관계

User 1:N Pass

Product 1:N Pass

Order 1:1 Pass

Pass 1:N SeatUsage

Pass 1:N LockerUsage
